do one more thing

你好，我是"三个程序猿的日常"的团队的一位开发者，目前研究的方向是数据挖掘和数据可视化，是未来立志做一名数据科学家。

人生格言是：do one more thing。

其实自己与计算机的缘分是在小学三年级才开始的，那时候还只知道打游戏，不同ACM大佬们那么牛逼。

高中时参加化学竞赛，曾去南大集训过，那时候就对周志华老师有所耳闻，虽然大学选择的是化学，但是奠定了我之后往机器学习的方向去奉献自己一点努力，经过很长时间的打磨，我坚信自己是对艺术和AI的热爱是真正的，

现在人工智能的浪潮来袭。简单说来，传统软件，大部分都是逻辑编码，每一步怎么做，最后如果判断，都靠人写命令告诉它。清晰，简明，优雅。

而人工智能，给与机器一定的自我学习能力。从数据中学习，就像人类擅长的那样，从错误中吸取经验，不断迭代，降低犯错的概率。数据科学，给与我们新的洞察力。不也很好玩吗？

最近我在做什么？

  • 和我的队友们一起愉快的开发一个创业项目。
  • 正在不断学习大数据和机器学习的知识
  • 在leetcode上刷题提升自己的编码能力

已完成：

  • 当然是顺利完成个人网站的开发和维护咯
